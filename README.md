# Portfolio

A modern and responsive portfolio website built with **Next.js**, **TypeScript**, and **Tailwind CSS**.  
It showcases my projects, skills, and experience as a software developer, with full internationalization support via **next-intl**.

[![Next.js](https://img.shields.io/badge/Next.js-000000?style=for-the-badge&logo=next.js&logoColor=white)](https://nextjs.org/) [![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=for-the-badge&logo=typescript&logoColor=white)](https://www.typescriptlang.org/) [![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=for-the-badge&logo=tailwind-css&logoColor=white)](https://tailwindcss.com/) [![next-intl](https://img.shields.io/badge/next--intl-0A0A0A?style=for-the-badge&logo=react&logoColor=61DAFB)](https://github.com/amannn/next-intl)

## Features

- Multi-language support using `next-intl` (English and Portuguese-BR)
- Modular components built with **Next.js** and **TypeScript**
- Styled entirely with **Tailwind CSS**
- Dynamic routing and optimized structure
- Easy to extend with new projects or content


## Installation

Requires [Node.js](https://nodejs.org/) (version 10 or higher) and **npm**.

```bash
# Navigate to the project folder
cd <root-dir>

# Install dependencies
npm install

# Start the development server
npm run dev
```
